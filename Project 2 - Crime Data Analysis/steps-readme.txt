1, get Police Department Incidents Dataset(San Francisco) from https://data.sfgov.org/Public-Safety/-Change-Notice-Police-Department-Incidents/tmnf-yvry/data

2, Create hive table, by using hive.sql

3, runing sql in hive.sql from Hive shell.

4, running sql in pyspark folder by using pyspark shell

5, upload data into databrick cloud, and create a table named pdi. Then runing sql in sparksql.sql on databrick.  https://community.cloud.databricks.com/?o=6289929760174924#notebook/4352377016250780

6, use tableau desktop to coustom report, geolaction based, category based, data based etc. pdi.twb is a tableau workbook file.

7, push tableau report to tableau online.  https://us-east-1.online.tableau.com/#/site/shanggao/workbooks/265129/views
